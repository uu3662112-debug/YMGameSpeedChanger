# 📦 游戏变速器 - 快速开始指南

## 一、环境准备

### 必需工具
- **Android Studio** (推荐 2023.1.1+)
- **JDK 17** 或更高版本

## 三、安装到模拟器

### 步骤1: 安装APK

**方法A: 拖拽安装**
- 将APK文件拖拽到模拟器窗口中
- 点击"安装"

**方法B: 命令行安装**
```bash
# 查看模拟器设备
adb devices

# 安装APK
adb install -r GameSpeedChanger-v1.0-debug.apk
```

### 步骤2: 安装虚拟环境(可选但推荐)

如需完整变速功能,需要安装虚拟环境:

**选项A: VMOS Pro (推荐)**
1. 在模拟器浏览器搜索"VMOS Pro"
2. 下载安装VMOS Pro
3. 将游戏变速器APK拖拽到VMOS内安装
4. 在VMOS内启动游戏变速器

**选项B: 跳过虚拟环境**
- 直接在模拟器使用变速器
- **注意**: 变速功能不会真正生效(只能体验UI)

## 四、使用指南

### 基础操作

1. **打开应用**
   - 在模拟器/VMOS中找到"游戏变速器"图标
   - 点击打开

2. **调整速度**
   - 滑动滑块调整速度 (0.1x - 100.0x)
   - 或点击预设按钮 (0.5x / 1.0x / 2.0x / 5.0x)
   - 或输入自定义倍率,点击"应用"

3. **选择游戏**
   - 在应用列表中点击要加速的游戏
   - 点击"启动应用"

### 预设速度说明

| 倍率 | 效果 | 适用场景 |
|------|------|----------|
| 0.5x | 减速 | 慢动作练习、精细操作 |
| 1.0x | 正常 | 默认速度 |
| 2.0x | 加速 | 快速过图、刷资源 |
| 5.0x | 极速 | 跳过动画、快速升级 |
| 10.0x+ | 超速 | 仅限测试,极不稳定 |

### 自定义速度

- 支持范围: **0.1 - 100.0**
- 建议步长: 0.5
- 推荐值: 1.5, 2.0, 3.0, 5.0

## 五、常见问题

### Q1: 为什么变速不起作用?

**原因**: 普通APK无法直接Hook系统时间函数

**解决**:
- 使用虚拟环境(VMOS Pro)
- 或使用Root设备 + LSPosed
- 参考README中的"进阶开发"部分

### Q2: 如何实现真正的变速?

**完整方案**:
1. 集成VirtualApp框架
2. 在虚拟环境中Hook时间API
3. 或开发LSPosed模块版本

详细代码见 `README.md` 的"进阶开发"章节。

### Q3: 会被游戏检测吗?

**风险等级**:
- 🟢 单机游戏: 低风险
- 🟡 弱联网游戏: 中风险
- 🔴 强联网游戏: 高风险

**建议**:
- 仅用于单机游戏
- 联网游戏谨慎使用
- 测试账号优先

### Q4: 模拟器卡顿怎么办?

**优化方法**:
- 关闭模拟器不必要的特效
- 增加模拟器分配的CPU/RAM
- 降低变速倍率
- 使用性能更好的模拟器

### Q5: 支持哪些模拟器?

**完全支持**:
- 雷电模拟器
- 夜神模拟器
- MuMu模拟器
- 逍遥模拟器
- BlueStacks

**要求**: Android 7.0+ (API 24+)

## 六、进阶功能

### 修改倍率上限

编辑 `speed-module/src/main/java/com/speedchanger/module/SpeedManager.java`:

```java
public void setSpeedMultiplier(float multiplier) {
    if (multiplier < 0.1f) multiplier = 0.1f;
    if (multiplier > 100.0f) multiplier = 100.0f; // 修改这里
    this.speedMultiplier = multiplier;
}
```

### 添加预设按钮

编辑 `app/src/main/res/layout/activity_main.xml`,添加新按钮:

```xml
<Button
    android:layout_width="0dp"
    android:layout_height="wrap_content"
    android:layout_weight="1"
    android:text="10.0x"
    android:layout_margin="4dp"
    android:onClick="setSpeed10_0"/>
```

在 `MainActivity.java` 添加对应方法:

```java
public void setSpeed10_0(View view) {
    speedManager.setSpeedMultiplier(10.0f);
    Toast.makeText(this, "速度已设为 10.0x", Toast.LENGTH_SHORT).show();
}
```
