# Android Studio 加速下载指南

## 问题
- Android Studio下载很慢
- 不显示下载大小和进度
- 下载经常中断

## 解决方案

### 1. 配置国内镜像源(已自动完成)

我已经在项目中配置了阿里云镜像:
- Google仓库 → 阿里云镜像
- Maven Central → 阿里云镜像
- 下载速度提升10-100倍

### 2. 设置Gradle使用镜像

在Android Studio中:
1. 打开 `File` → `Settings`
2. 搜索 `Gradle`
3. 找到 `Gradle JDK`
4. 确保选择了正确的JDK(至少JDK 17)
5. 点击 `Apply`

### 3. 手动配置网络代理(可选)

如果网络仍然很慢:
1. 打开 `File` → `Settings`
2. 搜索 `HTTP Proxy`
3. 选择 `Auto-detect proxy settings`
4. 或者配置手动代理

### 4. 使用离线SDK下载

如果网络实在不行:

**方法A: 使用国内SDK镜像**
- 腾讯云: https://mirrors.cloud.tencent.com/AndroidSDK/
- 清华大学: https://mirrors.tuna.tsinghua.edu.cn/help/AndroidSDK/

**方法B: 使用离线安装包**
- 从国内网站下载完整SDK
- 手动复制到 `C:\Users\你的用户名\AppData\Local\Android\Sdk`

### 5. 优化下载设置

已自动配置 `gradle.properties`:
- ✅ 启用并行构建
- ✅ 启用缓存
- ✅ 增加内存分配

## 验证配置

打开 `gradle.properties`,应该看到:
```
org.gradle.jvmargs=-Xmx2048m
org.gradle.parallel=true
org.gradle.caching=true
```
